CSV Format Guide

  • Each row = one complete heartbeat
  • Exactly 187 comma-separated floats per row
  • No header row needed (numbers only)
  • Scientific notation OK (e.g. 9.29E-01)
  • Multiple rows allowed — first row is analyzed

5 Arrhythmia Classes

  • Supraventricular — Above ventricles
  • Ventricular — In ventricles (high risk)
  • Fusion — Mixed beat origin
  • Unknown — Unclassified
